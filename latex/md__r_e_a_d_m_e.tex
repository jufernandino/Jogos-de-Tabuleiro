\doxysection{Descrição do projeto}\label{md__r_e_a_d_m_e_autotoc_md1}
O projeto Jogos de \doxyref{Tabuleiro}{p.}{class_tabuleiro} é uma aplicação desenvolvida como trabalho final da disciplina de Programação e Desenvolvimento de Software II (PDS II) do curso de Sistemas de Informação na UFMG. Este projeto implementa um sistema para jogar diferentes jogos, proporcionando funcionalidades cadastro de jogadores, exclusão de jogadores, estatísica dos jogadores e a listagem dos jogadores. Os jogos disponibilizados para serem jogados são\+:
\begin{DoxyItemize}
\item Campo Minado
\item \doxyref{Reversi}{p.}{class_reversi}
\item Tic Tac Toe
\item Jogo da Memória
\item Lig4
\end{DoxyItemize}\doxysection{Equipe}\label{md__r_e_a_d_m_e_autotoc_md2}
