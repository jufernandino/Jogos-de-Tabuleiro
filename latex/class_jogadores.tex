\doxysubsection{Descrição detalhada}
Classe para representar os jogadores. 

A classe "{}\+Jogadores"{} fornece uma interface básica para cadastrar jogadores, pesquisar por jogador, fazer login, remover jogador, salvar dados de vitória e derrota de cada jogador em estatisticas, visualizar as estaticas, mostrar o ranking de cada jogo, carregar jogadores e ordenar os jogadores por apelido ou por nome. Tudo isso é realizado com persistência de dados em arquivo.
