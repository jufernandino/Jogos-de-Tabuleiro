\doxysubsection{Descrição detalhada}
Classe base para representar um tabuleiro de jogo. 

A classe "{}\+Tabuleiro"{} fornece uma interface básica para criar, imprimir e manipular um tabuleiro de jogo. É uma classe base para o tabuleiro dos jogos.
