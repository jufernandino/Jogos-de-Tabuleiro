Jogos de Tabuleiro 1.0
O projeto Jogos de Tabuleiro é uma aplicação desenvolvida como trabalho final da disciplina de Programação e Desenvolvimento de Software II (PDS II) do curso de Sistemas de Informação na UFMG. O objetivo deste projeto é desenvolver um sistema usando o paradigma de orientação a objetos para implementar diferentes jogos de tabuleiro, proporcionando funcionalidades de cadastro de jogadores, exclusão de jogadores, estatística dos jogadores e a listagem dos jogadores. Os jogos disponibilizados são: Campo Minado, Reversi, Tic Tac Toe, Jogo da Memória e Lig4.

Descrição detalhada

Classe que representa o jogo Campo Minado.

A classe "Campo Minado" é uma classe herdeira de classe base "Tabuleiro" e implementa a lógica específica do jogo de Campo Minado, incluindo a verificação de condição de vitória (função que sobrescreve a da classe base). Além disso inclui métodos próprios como: inicialização do tabuleiro, revelar o conteúdo de cada casa, validação de jogadas, conferir se uma bomba foi escolhida, e a jogabilidade em si.

Esta função sobrescreve o método "confereGanhador" da classe base "Tabuleiro" e implementa a lógica para verificar se o jogador venceu na partida de Campo Minado.

Condição de vitória em um jogo de Campo Minado:

Valores Retornados
int0
int1

Esta função confere se ainda existe alguma casa + no tabuleiro que não foi escolhidas pelo jogador e, além disso, averigua se ela não é correspondentes a uma bomba. Caso ainda exista, significa que tem jogadas pendentes a serem realizadas. Em caso negativo, o jogo acabou e nenhuma bomba foi encontrada. Ou seja, o jogador ganhou.

Retorna
0, caso ainda exista jogadas possíveis.
1, caso o jogo tenha acabado.
